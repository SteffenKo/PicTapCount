# PROJECT.md

## Product Name

PicTapCount

## Vision

PicTapCount is a fast offline mobile application for counting objects on photos.

The first use case is counting piglets in agricultural environments.

Users can:

* Take a photo
* Import a photo
* Tap objects on the image
* See counting points
* Undo points
* Reset counting
* Save results
* Reopen previous counts

## Design Goals

* Minimalistic
* Fast
* Offline-first
* Easy to use with one hand
* Optimized for field work

## MVP Features

### Home Screen

* Quick access to camera
* Recent photos list
* Thumbnail preview
* Title
* Count result
* Date

### Counting Screen

* Image display
* Zoom and pan
* Tap to count
* Visual point marker
* Counter display
* Undo
* Reset

### Storage

* Local only
* No cloud
* Persist images and count data
